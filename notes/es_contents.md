# 基本概念与安装部署

## 基本概念

> 此处使用数据库的概念进行类比

| 概念 | 类比 |
| ---- | ---- |
| 索引(indices) | 数据库（Database）|
| 类型(type) | 表(Table) |
| 文档(Document) | 行(Row )|
| 字段(Field) | 列(Columns) |

通过这个类比可以看到，es最基础就是索引，所有数据都依赖于它。
