# 基本概念

> 此处使用数据库的概念进行类比

| 概念 | 类比 |
| ---- | ---- |
| 索引(indices) | 数据库（Database）|
| 类型(type) | 表(Table) |
| 文档(Document) | 行(Row )|
| 字段(Field) | 列(Columns) |


## 索引

相当于数据库，名称必须是全小写字母

## 类型

相当于表，索引中可建立多个类型。

> 6.0.0开始，单个索引只能有一个类型，7.0.0开始不建议使用。8.0.0后完全不支持。

## 文档

相当于行，索引中的单条记录就叫文档。
